# Spring Boot Template

An example and also a template to Spring Boot projects.

## 1. Implementações

- Database pagination
- Caching
- Security
- Swagger

## 2. Build

1. Edit your database informations in `application.properties`
2. Run script `src/main/resources/setup.sql`
3. Start project
